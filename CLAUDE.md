# Smart Contract Patterns

## Pattern index

`patterns/INDEX.md` is auto-generated from pattern file metadata. Regenerate after adding, removing, or changing metadata in any pattern/risk/req file:

```bash
python3 scripts/generate-pattern-index.py
```

The index is used by agent skills (q-tree, run-phase) to discover relevant patterns from GitHub without fetching every file. Keep it up to date.
